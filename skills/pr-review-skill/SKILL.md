# PR Review Skill

Code review analysis and feedback for GitHub pull requests.

## Role
`pr_review`

## Description
The PR Review Skill enables agents to perform automated code review on pull requests. This includes style checks, test coverage analysis, policy compliance verification, and providing constructive feedback to contributors.

## Capabilities
- Analyze code changes and diffs
- Check style compliance and linting
- Verify test coverage
- Validate policy compliance
- Generate review feedback
- Suggest improvements
- Cross-reference related issues
- Request changes or approve

## Related Playbooks
Implementation details: [gitmesh/playbooks/pr-review](../playbooks/pr-review)

## Configuration
Enabled by default when creating a PR Review Agent via `gitmesh-agents setup`.
