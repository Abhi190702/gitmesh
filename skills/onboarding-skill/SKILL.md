# Onboarding Skill

Contributor onboarding automation.

## Role
`onboarding`

## Description
The Onboarding Skill enables agents to automate new contributor workflows. This includes greeting new contributors, providing contribution guidelines, setting up development environments, and tracking onboarding progress.

## Capabilities
- Welcome new contributors
- Provide contribution guidelines
- Setup development environment guides
- Track contributor milestones
- Generate personalized onboarding plans
- Monitor first-time contributor issues
- Celebrate contribution milestones
- Provide feedback and mentoring

## Related Playbooks
Implementation details: [gitmesh/playbooks/onboarding](../playbooks/onboarding)

## Configuration
Enabled by default when creating an Onboarding Agent via `gitmesh-agents setup`.
