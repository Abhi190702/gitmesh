# Community Skill

Community engagement and discussion monitoring.

## Role
`community`

## Description
The Community Skill enables agents to monitor and facilitate community discussions, respond to common questions, and encourage contributor participation on GitHub Discussions, Discord, and other community channels.

## Capabilities
- Monitor GitHub Discussions
- Answer frequently asked questions
- Recognize and acknowledge contributors
- Facilitate discussions
- Cross-post useful content
- Community metrics and reporting
- Event announcements
- Discussion moderation assistance

## Related Playbooks
Implementation details: [gitmesh/playbooks/community](../playbooks/community)

## Configuration
Enabled by default when creating a Community Agent via `gitmesh-agents setup`.
