> **Phase 1** — This feature is under active development.

This guide will cover:

- Configuring the forge provider (GitHub) in Project Settings
- Setting up the webhook URL in your GitHub repository
- Bidirectional issue sync between GitMesh Agents and GitHub Issues
- Viewing sync status in the Forge Sync page

## Prerequisites

- A running GitMesh Agents instance
- A GitHub repository you want to connect
- Repository admin access (to configure webhooks)
